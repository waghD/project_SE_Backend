Projekt SE Backend
=============

Folgende Schritte sind zum starten des Programs notwendig

# Schritt 1

```python
py manage.py migrate
```

# Schritt 2

```python
py manage.py makemigrations logistics
```

# Schritt 3

```python
py manage.py createsuperuser
```

# Schritt 4

```python
py manage.py runserver
```
